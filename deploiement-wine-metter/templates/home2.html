API Documentation

Bienvenue dans la documentation API de notre application de Machine Learning sur Quality Wine.

Cette API vous aidera à choisir un vin de meilleure qualité. Par conséquent, vous apprécierez mieux votre verre de vin pour la soirée.

Endpoint 1 : Entrez vos données

GET / POST : https://wine-o-flavs.herokuapp.com/

Ce critère vous permet de faire une prédiction de vin de qualité avec les caractéristiques que vous renseignez (ex : Alcool, Chlorures ; pH)

Comment demander ce point de terminaison ?

Il vous suffit de cliquer sur le bouton « predict » après avoir saisi les données (11 caractéristiques).
Le bouton prredict qui utilisera la méthode POST. Il utilisera la racine ("https://wine-o-flavs.herokuapp.com/")
La racine applique en fait le modèle de machine Learning.

Exemple : Saisissez les données et cliquez sur « Predict »

Il renvoie par exemple un message 'Ce vin est de qualité : {} / 10'

Endpoint 2 : API request

POST : https://wine-o-flavs.herokuapp.com/predict_api

Ce point de terminaison vous permet de faire une prédiction de vin de qualité avec les caractéristiques que vous envoyez/postez.

Comment demander ce point de terminaison ?

Vous devez demander ce point de terminaison à l'aide d'une méthode POST avec un contenu JSON. Votre JSON doit contenir les 11 éléments requis (ex : Alcool, Chlorures en format float).

Example 1: cette commande sera à effectuer dans le terminalcurl -i -H "Content-Type: application/json" -X POST -d '{"input": [[6.6,0.16,0.4,1.5,0.044,48,143,0.9912,3.54,0.52,12.4]]}' https://wine-o-flavs.herokuapp.com/predict_api

Il renvoie un JSON comme ceci : 

          
  {'predict': 7.0}
